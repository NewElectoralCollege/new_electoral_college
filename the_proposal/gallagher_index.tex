    This is the \LaTeX{} version of the formula for the Gallagher Index.
    \begin{align}
        LSq &= \sqrt{\frac{1}{2}\sum_{i=1}^{n} {(V_i - S_i)^2}}
    \end{align}
    Verbatim:
    \begin{verbatim}
        LSq &= \sqrt{\frac{1}{2}\sum_{i=1}^{n} {(V_i - S_i)^2}}
    \end{verbatim}
    Uses the amsmath package.
